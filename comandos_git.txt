git init . 		 -> Cria um repo na pasta atual
git status 		 -> Mostra o status do trabalho (de cada arq)
git add nome_arquivo     -> Adiciona arquivo para ser comitado
git commit -m "msg"	 -> Faz commit do que estava pra ser comitado
git log 		 -> Mostra todos o histórico de commit

## FLUXO DE TRABLHO
1. Criar o diretório do projeto
2. Iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3.
